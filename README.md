<h1 align="center">
  Sonnet
</h1>

<h4 align="center">
  🍃 Have a break with William Shakespeare's sonnets from the terminal.
</h4>

### INSTALLATION

```bash
npm install --global sonnet
```

### HOWTO
```
Usage

    $ sonnet <option>

    Options

        --random, -r      Display a random piece
        --version, -v     Display installed version
        --help, -h        Display help tips 
```
